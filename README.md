# CIS (Color Image System) Specification

## 1. 概要
CISは、16ビットインデックスカラーとランレングス圧縮を用いたテキストベースの画像表現システムです。
バイナリではなく「人間が読める・書ける」テキスト形式で画像を定義し、データ容量の極小化を目指します。

## 2. ファイル形式
- **拡張子**: `.cid` (Color Image Data)
- **文字コード**: UTF-8推奨

## 3. 構造
ファイルはヘッダー領域とデータ領域で構成されます。

### ヘッダー (Header)
行頭にキーワードを記述します。
- `TITLE:` 作品名
- `AUTHOR:` 作成者
- `DATE:` 作成日
- `WIDTH:` 画像の横幅 (px)

### データ (Data)
ピクセルデータをカンマ(`,`)区切りで記述します。
- **単一ピクセル**: `[0-65535]` (例: `63488`)
- **ランレングス圧縮**: `[色番号]*[連続数]` (例: `0*10`)
- **色空間**: RGB565 (16bit)

## 4. 色変換ルール (RGB565)
24bitカラー(RGB)を16bit数値に変換する際のビット割り当て：
- Red:   5bit (bit 11-15)
- Green: 6bit (bit 5-10)
- Blue:  5bit (bit 0-4)
